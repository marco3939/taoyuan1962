## 🆕 v7.1 核心轉折：機場消費 → 航空博物館 + 一條龍服務

### 主收益模組：過境 6hr+ 黑貓一條龍 4hr 套裝
把桃機過境旅客原本在機場免稅店的「分散消費」轉化為航空博物館的「主題深度體驗 + 整套消費」，搭配觀光署 NT$600 雙券分立。

| 階段 | 時間 | 內容 |
|---|---|---|
| 1. 出境通關+寄物 | 0:00–0:35 | Fast Track + 寄存登機箱 |
| 2. 機捷接駁 | 0:35–1:00 | T1=A12 / T2=A13 → A16 橫山 + 接駁專車 |
| 3. **★ 黑貓基地 4 站體驗** | 1:00–2:30 | **90 分核心 — 換裝簡報→沉浸→下午茶→打卡家書** |
| 4. 接駁回機場 | 2:30–2:55 | 同路徑 |
| 5. 機場緩衝 | 2:55–4:00 | NT$600 觀光署禮券於此使用 |
