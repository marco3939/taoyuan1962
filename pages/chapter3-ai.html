CHAPTER 3.4 · AI + SYSTEM-LEVEL GUARDRAILS

AI 互動模組 + 系統級 Guardrails 防禦架構

首頁 / 四維融合 / RAG + Guardrails 三層 + air-gapped · P14

回首頁
v6 AI 升級重點：取消「人工審核」依賴，改採 Guardrails 三層系統級防禦（L1 輸入過濾 / L2 RAG 強制錨定 / L3 輸出過濾）+ 本地 air-gapped 部署。Corpus 物理隔離，不對外送第三方 API。對應風險矩陣 R7 / R10 / R11 紅區三項緩解。

系統架構 · 本地部署 + Air-gapped

層級規格
LLM 模型Llama 3.1 70B / Qwen 2.5（本地部署，無雲端 API）；量化至 INT8
EmbeddingBGE-M3（多語） / Cohere Embed（中文優化）；向量維度 1024
向量 DBQdrant 自建集群（本地端）；分區 Air-gapped corpus
GPU2× NVIDIA A100 80GB（地端機房）/ 推論服務 RAG 流程
網路DMZ 隔離 + WAF + 對外 API 經 reverse proxy + 速率限制

Guardrails 三層系統級防禦（v6 NEW · 取代「人工審核」）

L1 · INPUT FILTER

L1 輸入過濾

技術：Llama Guard 3 / NeMo Guardrails Input Rails

防禦目標：Prompt Injection、越獄指令、敏感主題誘導

L2 · RAG ANCHORING

L2 RAG 強制錨定

技術：Citation-required + Score Threshold（<0.7 拒答）

防禦目標：幻覺、未授權史實、虛構人物對話

L3 · OUTPUT FILTER

L3 輸出過濾

技術：Output Rails + 史學白名單詞庫

防禦目標：歷史虛無主義、PII 洩漏、敏感擊落數字

輸入 → L1 Input Rails → LLM + RAG → L2 Citation 強制 → L3 Output Rails → 輸出

資安、隱私治理 + 禁止項

項目做法
個資/聲紋個資法第 6 條敏感類別不蒐集；聲紋當場刪除不存檔
對話日誌30 天 hash 化後刪除
Corpus 治理口述史知情同意 + 書面授權 + 撤回機制
稽核（v9.0 強化）每季滲透/紅隊測試；ISO 27001 框架對標；導入自動化資安監控與虛擬化資源調度工作流，確保本地端 AI 機房長期維運 OPEX 降至最低
危機應變AI 異常 SLA：24h 鎖定模型 / 72h 對外公告
禁止項(1) 自由生成歷史人物對話（必須 RAG + Citation）；(2) 生成具體擊落數字、未公開隊員姓名；(3) 對外 API 傳送 corpus（air-gapped）

AI 四種應用

AI-A

個人化導覽

技術：推薦演算法 + LLM
場景：行前 Chatbot、動態動線、即時多語（中英日韓）

AI-B

對話式角色（RAG）

技術：RAG + 開源 LLM
場景：虛擬黑貓老兵、虛擬眷屬、史學顧問互動

AI-C

情緒適應

技術：情緒辨識（可選）
場景：VR 沉浸強度動態調整、無障礙降階

AI-D

出館回顧

技術：生成式摘要
場景：體驗摘要、社群短影音生成、回訪推薦

★ AI 倫理揭露 · 強制邊界

項目做法
模型選用本地部署 Llama 3.1 70B / Qwen 2.5（air-gapped 不對外）[L3]
Corpus已解密文件 + 見證者授權口述史（≥30 hr）+《一把青》原著（已授權）

幻覺緩解Guardrails 三層（L1/L2/L3）取代 v5 人工審核 + 介面警示「AI 生成，僅供參考」
透明度每段 AI 回覆附「原始史料連結」與「來源錨定編號」
禁止項禁止 AI 自由生成歷史人物對話（必須 RAG + Citation）、禁止生成擊落數字、未公開隊員姓名；對外 API 傳送 corpus 嚴禁（air-gapped）

RAG 互動範例 · 透明來源錨定

使用者：「你怕嗎？」
AI 回覆（基於 corpus）：
「飛 7 萬呎以上，座艙壓力比月球表面還低。你問我怕？恐懼變成程序。」
來源錨定：見證者口述史 #017，1965 任務後採訪片段 [L1]
警示標籤：AI 生成內容，僅供參考。原始錄音 [連結]
